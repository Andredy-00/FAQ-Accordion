What is Frontend Mentor, and how will it help me?

Frontend Mentor offers realistic coding callenges to help developpers improve their frontend coding skills with projects in HTML, CSS and JavaScript. Its suitable for all levels and ideal for portfolio building.

Is Frontend Mentor Free?

Yes, it is free and an excellent help for practice, i highly recommend it

Can i use Frontend Mentor projects in my portfolio?

Of course you can use these projects for your portfolio, there are excellent challenges

How can i get help if Im stuck on a challenge?

The best place to get help is inside Frontend Mentor's Discord community. There's a help channel where you can ask questions and seek support from other community members.
